# Capacitor Native Assets

Coloque aqui os arquivos de ícone e splash screen para gerar os assets nativos:

- `icon.png` — 1024x1024px (ícone do app)
- `splash.png` — 2732x2732px (tela de splash)

Para gerar os assets nativos, execute:

```bash
npx @capacitor/assets generate --iconBackgroundColor '#0a0a0a' --splashBackgroundColor '#0a0a0a'
```
